# Lib

Utilitaires génériques et helpers sans dépendance au UI.

Contenu typique :
- `formatters/` : prix, dates, textes.
- `validators/` : schémas Zod et règles.
- `constants/` : valeurs globales (dev/prod, routes, clés).

Objectif : centraliser les fonctions transverses pour éviter la duplication.
