# Hooks

Hooks React personnalisés, découplés des composants.

Exemples :
- `useCart()` : gestion locale du panier.
- `useAuth()` : état utilisateur, tokens, redirections.
- `useDebounce()` : antirebond pour la recherche.
- `usePrice()` : formatage et logique d’affichage des prix.

But : partager de la logique réutilisable entre composants et features.
