# Components

Composants UI réutilisables et indépendants du métier. Séparer par types :
- `ui/` : atomes et molécules (bouton, input, modal, dropdown, tooltip).
- `layout/` : structures globales (header, footer, navbar, container).
- `forms/` : champs et helpers de formulaires (intégration RHF/Zod).

Objectif : construire l’interface de manière cohérente et réutilisable.
