# Types

Types et interfaces TypeScript partagés.

Fichiers suggérés :
- `product.ts` : modèle produit (id, nom, prix, variantes, images).
- `cart.ts` : lignes de panier, totaux, coupons.
- `user.ts` : profil, adresses, rôles.
- `order.ts` : commande, statuts, paiements.

But : type-safety et cohérence à l’échelle du projet.
